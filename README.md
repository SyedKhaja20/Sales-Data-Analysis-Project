# 📊 Sales Data Analysis Project

## 🔍 Overview

This project analyzes a retail sales dataset to uncover key business insights related to sales performance, customer behavior, and profitability.

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Jupyter Notebook

---

## 📈 Key Analysis Performed

* Data Cleaning & Preprocessing
* Monthly & Yearly Sales Trends
* Top Products Analysis
* Region-wise Sales & Profit
* Loss-Making Products Identification
* Correlation Analysis

---

## 💡 Key Insights

* Technology category generated the highest revenue
* Certain products contributed disproportionately to total sales
* Some products incurred losses despite high sales
* Discounts negatively impacted profit margins
* Sales showed seasonal trends over time

---

## 📂 Dataset

* Superstore Sales Dataset (Kaggle)

---

## 🚀 Conclusion

The analysis provides actionable insights that can help businesses improve profitability, optimize inventory, and make data-driven decisions.
